Iterated Self-Evaluation Impossibility for Large Language Models

J. Suh
IEEE Access  ·  In preparation, 2026

Modern LLMs are rarely run once. Chain-of-thought, self-critique, and agentic loops all run a model repeatedly on its own previous output, trusting that iteration will settle on a stable answer the model endorses. This paper asks a precise question: can a model decide, of its own reasoning loop, that the loop has converged on an answer it endorses?

The answer is no, and quantifiably so. Modelling a single forward pass as a computable function and the loop as the partial computation it induces, I show that deciding convergence to a self-endorsed answer is Σ₂-complete, and in particular undecidable even given an oracle for the halting problem. Self-evaluation is therefore strictly harder than halting: even a machine that could detect non-termination still could not decide it. A self-referential argument sharpens the point, any evaluator you could actually implement as a prompt can be effectively falsified on an input built from its own description.

The result concerns the unbounded-iteration idealisation: a finite deployed model stays decidable, but the verification horizon grows without bound.
