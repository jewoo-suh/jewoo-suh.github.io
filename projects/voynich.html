Voynich Decode Project

Closed  ·  Solo  ·  Statistical Analysis  ·  Computational Linguistics

The Voynich Manuscript is a 15th-century illustrated codex in an unknown script. A century of attempted decipherment has produced nothing. The standard hypothesis is that it encodes a language or cipher. This project asked a different question: do the statistical properties of the text actually require a language to explain?

A ~200-unit combinatorial generative procedure, with no semantics or intended meaning, just glyph templates and positional rules, reproduces every measured structural property of the manuscript: character distributions, word-length profiles, positional constraints, inter-word entropy. If content-free generation is sufficient to produce statistically identical text, there is no evidential basis for invoking a cipher. The manuscript probably says nothing.

The ridge plot above shows Voynichese character distributions against natural languages and constructed controls. It clusters with the synthetic controls. Project closed.
